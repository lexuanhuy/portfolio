# Modern Portfolio

A high-performance, responsive personal portfolio built with React 19, TypeScript, and Tailwind CSS 4.

## 🚀 Features

- **React 19**: Leveraging the latest React features for optimal performance.
- **TypeScript**: Full type safety for a better developer experience.
- **Vite 7**: Ultra-fast build and development server.
- **Tailwind CSS 4**: Modern utility-first styling.
- **React Router 7**: Robust routing for single-page application feel.
- **Cyber-themed UI**: Unique aesthetics with glow effects and custom animations.
- **Pre-commit Hooks**: Automated linting and formatting with Husky and lint-staged.

## 🛠️ Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Build Tool**: Vite
- **Routing**: React Router
- **Formatting & Linting**: ESLint, Prettier, Husky, lint-staged
- **Package Manager**: pnpm

## 📦 Getting Started

### Prerequisites

- Node.js (v18+)
- pnpm (v9+)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Start the development server:
   ```bash
   pnpm dev
   ```

### Scripts

- `pnpm dev`: Start development server.
- `pnpm build`: Build for production.
- `pnpm lint`: Run ESLint checks.
- `pnpm preview`: Locally preview the production build.

## 📂 Project Structure

- `src/components/ui`: Atomic UI components.
- `src/components/sections`: Major page sections.
- `src/pages`: Top-level page components.
- `src/types`: TypeScript interfaces and types.
- `src/assets`: Images, icons, and global styles.
